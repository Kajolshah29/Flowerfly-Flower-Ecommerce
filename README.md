# Flowerfly - Flower E-commerce

## Description
An E-Commerce plant shop app designed for a seamless and hassle-free plant shopping experience. Built with Firebase, it ensures fast, secure, and reliable transactions. Featuring a modern and minimal design, the app offers a visually appealing and user-friendly interface for effortless browsing and purchasing.

## Technology Used
- Kotlin
- Android
- Gradle
- Firebase Auth and Direbase Realtime Database
